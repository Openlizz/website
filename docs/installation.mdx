# Installation

You can install the `lizz` command-line-interface (CLI) in several ways depending on your operating system.

## Linux and macOS

<Tabs>
<TabItem value="homebrew" label="Homebrew" default>

```
brew install openlizz/tap/lizz
```

</TabItem>
<TabItem value="bash" label="Bash">

```
curl -s https://raw.githubusercontent.com/openlizz/lizz/main/installation/lizz.sh | bash
```

</TabItem>
</Tabs>

## Windows WSL

:::caution

Lizz is not yet properly tested on Windows. 

:::

```
curl -s https://raw.githubusercontent.com/openlizz/lizz/main/installation/lizz.sh | bash
```

---

You can find all the executable files in the [release page](https://github.com/Openlizz/lizz/releases).
